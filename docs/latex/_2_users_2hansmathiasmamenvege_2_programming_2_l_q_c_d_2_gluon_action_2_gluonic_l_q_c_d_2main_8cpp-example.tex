 This will generate a .json file called {\ttfamily config\+\_\+\+G\+L\+A\+C\+Test\+Run.\+json} located in a folder called input. The system will be of size $ 16^3 \times 32$, and have $\beta = 6.0$. It will run $N_\matrm{th} = 10^5$ thermalization steps, generate $N_\matrm{cfg} = 100$ configurations, and flow each of the \$ $N_\matrm{flow} = 1000$ a steps with a step size of $\epsilon_\matrm{flow} = 0.01$. The name of the run will be G\+L\+A\+C\+Test\+Run.
