Documentation for GLAC

GLAC is a program which generates gauge configurations and has the possibility of applying gradient flow on them as well.

It takes input in the form of .json files generated by createJobs.py, a python2 command line interfacing tool. createJobs.py will create relevant folders for the run. In the case that we are running on an HPC cluster using Slurm or Torque, it will directly initiate the run by submitting the jobs unless –dryrun is specified.

See Quickstart to GLAC for a short guide in how to use GLAC.
